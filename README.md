# ML Regression Basics

This repository is the starting point for learning clean machine learning project structure.

## Goal
Build a regression project step by step, beginning with environment setup, project structure, and script-based workflow.

## Project Structure

- `src/` contains Python scripts
- `data/` contains datasets
- `notebooks/` contains exploration notebooks
- `tests/` contains tests

## Environment

Create the environment with:

```bash
mamba create -n ml-regression python=3.11
conda activate ml-regression
